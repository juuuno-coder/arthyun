## 결함 리포트 (Defect Report)

### [Critical] 모바일 메뉴 작동 불가 (TC015)
- **설명**: 모바일 뷰포트에서 상단 'MENU' 텍스트를 클릭해도 아무런 반응이 없습니다.
- **원인**: `src/app/layout.tsx`에 `MENU` 텍스트만 렌더링되어 있고, 클릭 이벤트 핸들러나 메뉴 컴포넌트(`MobileMenu`)가 연결되지 않았습니다.
- **해결 방안**: Client Component로 모바일 메뉴 로직을 분리하여 구현해야 합니다.

### [Minor] 이미지 로딩 및 업로드 UX
- 업로드 진행률 표시 부재 및 이미지 스켈레톤 UI 부재가 확인되었으나 기능 실행에는 문제가 없습니다.
