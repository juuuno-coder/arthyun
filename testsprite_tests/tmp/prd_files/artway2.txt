개선 및 보완 필요 사항 리스트 (Improvement List)
브라우저 테스트 및 코드 리뷰를 통해 확인된 보완 사항입니다.

1. 긴급 (Critical)
모바일 메뉴 미구현: 모바일 뷰(너비 768px 이하)에서 'MENU' 텍스트만 표시되고, 실제 클릭 시 아무런 동작을 하지 않습니다. (
src/app/layout.tsx
)
조치 필요: MobileMenu 클라이언트 컴포넌트를 만들어 햄버거 메뉴 및 드로어 네비게이션 구현 필요.
2. 권장 (Recommended)
메인 페이지 빈 상태(Empty State) 개선: 현재 DB에 is_main_slider: true인 전시가 없으면 단순히 텍스트만 표시됩니다.
조치 제안: 기본 이미지나 회사 로고 등을 배경으로 깔아 시각적 완성도 높이기.
이미지 로딩 최적화: 갤러리 특성상 이미지가 많으므로 next/image의 placeholder="blur" 처리를 자동화하거나 스켈레톤 UI를 추가하면 UX가 개선될 것입니다. (현재는 기본 로딩)
3. 관리자 기능 (Admin)
이미지 업로드 피드백: 이미지 업로드 시 진행률(Progress 바)이 없어 대용량 파일 업로드 시 멈춘 것처럼 보일 수 있습니다.
삭제 확인 UI: 현재 브라우저 기본 confirm()을 사용 중입니다. ShadCN UI의 Dialog나 AlertDialog로 세련되게 변경하는 것을 권장합니다.
4. 기타 (Minor)
타이틀/메타데이터 세분화: 페이지별(archive, media)로 metadata를 동적으로 생성하여 SEO를 강화할 수 있습니다.
상점 페이지 예고: 단순 텍스트보다 "입점 문의" 버튼이나 "알림 받기" 같은 Call-to-Action(CTA) 버튼을 추가하면 좋습니다.
